## 🌊 Overview

**Titanic AI Platform** is a cinematic AI-powered SaaS-style web application that analyzes passenger survival patterns from the Titanic disaster dataset.

The platform combines:

- 📊 Executive Dashboard Analytics  
- 🤖 Machine Learning Survival Prediction  
- 📈 Model Explainability using SHAP  
- 💬 AI Data Assistant Chatbot  
- 🎬 Cinematic Deep-Sea Animated UI  

This project demonstrates real-world ML deployment skills with an interactive SaaS-style interface.
